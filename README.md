# Spring Boot + MongoDB Application

## Overview
This is a **Spring Boot** application connected to **MongoDB**. It is designed as part of the training program offered by **NextGen Institute of Technology**. The application demonstrates key concepts of full-stack Java development and database integration.

## Ownership

## Features
- RESTful API endpoints using Spring Boot
- MongoDB integration for data persistence
- Basic user management functionalities
- Designed for educational and training purposes

## Prerequisites
- Java 11+ or compatible
- Maven 3.6+
- MongoDB running locally or on cloud
- IDE (IntelliJ, Eclipse, VS Code, etc.)

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/<repo-name>.git

2. Navigate to the project folder:
cd <repo-name>

3. Build the project using Maven:
mvn clean install

4. Run the Spring Boot application:
mvn spring-boot:run

5.Access the API at:
http://localhost:8080
